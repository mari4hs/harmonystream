# 🎵 HarmonyStream

Uma landing page moderna de streaming musical com design responsivo e sistema de tradução multilíngue.


## ✨ Demonstração

**Acesse o site:** [HarmonyStream no GitHub Pages](https://mari4hs.github.io/harmonystream)


## 🚀 Funcionalidades

- 🌐 **Sistema de tradução** (Português/Inglês/Epanhol)
- 📱 **Design 100% responsivo** 
- 🎨 **Animações suaves** e efeitos visuais
- 🎵 **Player simulado** de música
- 💰 **Planos de assinatura** interativos


## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Grid, Flexbox, Animações
- **JavaScript** - Interatividade e lógica
- **Git & GitHub** - Versionamento e deploy
